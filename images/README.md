# Personal Photos for Academic Homepage

## 📸 照片文件说明

这些是占位图片文件，用于学术个人主页的个人照片显示。您可以将其替换为您的真实个人照片。

### 📁 文件列表

1. **profile-photo.svg** (400x400)
   - 用途：Hero section中的圆形头像
   - 格式：SVG矢量图
   - 建议：替换为您的个人照片（推荐JPG或PNG格式）

2. **profile-photo-large.svg** (800x800)
   - 用途：点击查看大图时的全屏显示
   - 格式：SVG矢量图
   - 建议：替换为高质量的个人照片

## 🔄 如何替换照片

### 方法1：直接替换（推荐）
1. 将您的个人照片重命名为：
   - `profile-photo.jpg` 或 `profile-photo.png`（用于头像）
   - `profile-photo-large.jpg` 或 `profile-photo-large.png`（用于大图）

2. 上传到 `images/` 文件夹

### 方法2：保持原文件名
如果您想保持SVG文件名，可以：
1. 将您的照片保存为 `profile-photo.jpg`
2. 修改HTML中的图片路径

## 📸 照片要求

### 头像照片 (profile-photo)
- **尺寸**：建议 400x400 像素（正方形）
- **格式**：JPG、PNG 或 WebP
- **背景**：建议使用纯色或渐变背景
- **内容**：专业的半身照或头像照
- **风格**：学术、专业、友好

### 大尺寸照片 (profile-photo-large)
- **尺寸**：建议 800x800 像素或更大
- **格式**：JPG（高质量）、PNG 或 WebP
- **质量**：高清晰度，适合全屏查看

## 🎨 推荐的照片风格

### 学术风格
- 正式的半身照或头像照
- 简洁的背景（白色、浅色或渐变）
- 专业的着装（商务休闲或正装）
- 友好自然的表情

### 技术风格
- 实验室或办公室环境
- 与研究领域相关的背景
- 展现专业性的姿态

## 📱 响应式适配

网站会自动适配不同设备：
- **桌面端**：显示圆形头像
- **平板端**：自适应圆形头像
- **手机端**：自适应圆形头像

## 🔧 技术说明

当前网站使用以下图片格式支持：
- **SVG**：矢量图，无限缩放（当前占位图片）
- **JPG**：适合照片，文件小，兼容性好
- **PNG**：支持透明背景，文件较大
- **WebP**：现代格式，文件小，兼容性较新

## ⚡ 优化建议

1. **图片压缩**：使用 TinyPNG 或类似工具压缩图片
2. **格式选择**：WebP > JPG > PNG（按兼容性和文件大小）
3. **尺寸适中**：头像建议不超过 200KB
4. **懒加载**：网站已实现图片懒加载功能

## 🔄 备份建议

1. 保留原始高分辨率照片
2. 创建多个尺寸的版本
3. 使用版本控制（如有需要）

**注意**：当前使用的是SVG占位图片，您可以直接替换为真实照片而无需修改代码。网站会自动适应新的图片格式。
